Follow the below steps to run the complete project:

1. First run main "Server" application which will connect to the database.

2. Now run the "Roulette Server" which will link the Customer Application to the main "Server".

3. Now start the "Customer Angular App", here customer can logIn by using his/her unique customer Id 
   and start playing.

4. To start the Admin UI application, run the "Admin UI App" and execute the "Home.jsp" file. 
   Now admin can 
	a) view the list of all registered customers.
	b) recharge the account balance of those customers.
	c) Register new customers.
